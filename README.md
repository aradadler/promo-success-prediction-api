# promo-success-prediction-api

Production-style project skeleton for a FastAPI-based machine learning inference service.

## Structure

- `src/app/`: API application package, schemas, config, and inference placeholders
- `data/`: utility scripts for synthetic or local data workflows
- `tests/`: starter test suite for inference logic and API smoke checks
